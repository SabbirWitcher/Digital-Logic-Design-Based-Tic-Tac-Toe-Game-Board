# Digital-Logic-Design-Based-Tic-Tac-Toe-Game-Board
This project uses digital logic design to implement a digital Tic Tac Toe Game board where players can press button to turn on LEDs with their corresponding LED colors inside the tic tac toe grid until one player wins or the game reaches draw.

The board controls the player turns and assignment of light colors automatically using digital logic and determines win condition to determine the winner.
 
Bi color LED is used to differentiate between players. Proteus was used to design the PCB and the run simulation before implementing the ICs into the PCB. When one player wins another bi colored LED changes lights up with that player's color to mark his win. For Draw condition a seperate LED will light up. Flip Flops are used to control the turns of the players.
